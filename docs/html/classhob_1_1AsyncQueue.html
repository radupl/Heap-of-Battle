Detailed Description

template<typename TYPE>
class hob::AsyncQueue< TYPE >

Thread safe STL queue for one consumer thread and one supplier thread.

It is used for updates that are being processed on other threads than the main one. In the case of multiple consumer threads one might empty the queue after the other checked it is not empty. This case is accepted as for the moment is not used this way.
