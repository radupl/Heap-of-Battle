Detailed Description

Static class handling chat commands regarding logging.

This class is present only if libplog.so is loaded.

void hob::LogManager::handleCommand ( const char *  command)
staticnoexcept

Handles a user's command regarding logging. Supported commands are:

  • ssl <uint8>: Sets the log severity level.
  • sfs <uint64>: Sets the log file size.
  • sfc <uint8>: Sets the log file count.
  • stm <bool>: Sets the log terminal mode.
  • sbs <uint64>: Sets the log buffer size.
  • gsl: Gets the log severity level.
  • gfs: Gets the log file size.
  • gfc: Gets the log file count.
  • gtm: Gets the log terminal mode.
  • gbs: Gets the log buffer size.
